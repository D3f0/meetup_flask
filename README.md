# Instalación

Para instalar flask:

```bash
pip install Flask
```

(Si no tenemos `pip`, bajarlo de [aquí](https://bootstrap.pypa.io/get-pip.py) y ejecutar `python get-pip.py`)


# Ejecución

```bash
python runserver.py
```

Luego abrir el navegador en http://127.0.0.1:5000/
